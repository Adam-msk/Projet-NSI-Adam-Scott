## Projet-NSI-Adam-Scott
Nous nous inspirons du jeu Flappy Bird pour notre projet. 
Dans ce jeu, le joueur incarne un oiseau soumis à la gravité.
En appuyant sur une touche, l’oiseau peut remonter vers le haut.
Il doit éviter les tuyaux qui apparaissent aléatoirement et qui défilent.
Le nombre de points est calculé en fonction du nombre de tuyaux évités.
